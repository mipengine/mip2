    1. 组件的 README.md 支持自动生成组件脚本地址
- 1.2.2
    1. cli 沙盒注入提示信息增加对应文件提示以及白名单申请引导

    1. 升级 validator 依赖 mip-component-validator 至 1.1.0，该版本全流程改为异步实现

    3. validate 命令增加 -w 参数校验 npm 白名单
    4. validate -c 命令增加组件所在白名单校验

- 1.1.10
