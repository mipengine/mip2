# 数据驱动所支持的表达式

在数据驱动机制当中提到了 `MIP.setData()`、`m-bind` 与 `m-text` 数据绑定都支持写一些表达式来完成一定程度的数据计算。比如：

```html
<div on="tap>MIP.setData({
    a: isActive ? 1 : 2,
    b: b + 1，
    c: [2, 1, 3].slice(1)
  })"
></div>
```

例子中的 `isActive ? 1 : 2`、`b + 1`、`[2, 1, 3].slice(1)` 都属于表达式。

这类表达式具有以下特点：

1. 仅能操作在 `<mip-data>` 中声明的变量；
2. 支持有限能力的运算规则；
3. 支持有限的方法；
